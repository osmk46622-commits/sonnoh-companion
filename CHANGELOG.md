# 📝 Changelog - Sonnoh AI Companion

## Version 1.0.0 (May 2026) - Initial Release ✨

### ✅ Features
- 6 personality variants (White, Blue, Red, Green, Yellow, Black)
- Custom 3D cat model with fluffy features
- Animated ears, tail, legs, walking
- Taming system with fish/salmon
- Care mechanics & emotional states
- Interactive dialogue system
- Special underground lab event
- Complete AI behavior system
- 7 animations (idle, walk, sit, pet, hug, ear_flick, sleep)

### 🎨 Content
- 13+ dialogue lines (variant-specific)
- 6 color variants with unique personalities
- Complete 3D model (16 bones)
- 64×64 texture system

### 🔧 Technical
- Bedrock Edition addon format
- Minimum engine version: 1.19.0
- Compatible with all Bedrock platforms
- Fully customizable JSON structure
