# 🚀 Quick Start - Sonnoh Companion

## 30-Second Installation

### Step 1: Download
👉 **[Get Latest Release](https://github.com/osmk46622-commits/sonnoh-companion/releases)**

### Step 2: Install
- Click the downloaded `.mcaddon` file
- Minecraft opens automatically
- Select your world
- Click "Add" ✅

### Step 3: Play!
```
/summon sonnoh:sonnoh
```

---

## Basic Controls

| Action | How |
|--------|-----|
| **Tame** | Hold fish + right-click |
| **Pet** | Right-click (empty hand) |
| **Follow** | Walk away (tamed only) |
| **Sit** | Right-click again |
| **Talk** | Type "Sonnoh" in chat |
